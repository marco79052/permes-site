ACCOUNT SAFETY

账号安全是我们的底线

从架构设计到运行时保护，Permes 在每一个环节都把账号安全放在第一位。

不读内存

不读取游戏内存、不注入进程、不修改客户端文件。仅通过截图观察屏幕像素。

全随机特征码

每次运行生成全新随机特征码，进程指纹不可追溯，无法被特征扫描识别。

代理启动

通过独立代理进程启动，主程序与游戏进程完全隔离，行为链路不可关联。

全局隐藏

运行时全局隐藏窗口与进程信息，任务管理器不可见，反作弊系统不可探测。

支持 DMA

支持 DMA 硬件模式，按键发送与截图采集完全独立于操作系统，物理级隔离。

按键保护

按键发送经过内核级保护，模拟真实硬件输入特征，注入痕迹从根源消除。

FAQ

常见问题

Permes 会读取或修改游戏内存吗？ +
不会。Permes 不读取游戏内存、不注入进程、不修改客户端文件。仅通过截图观察屏幕像素，从根源杜绝封号风险，法律层面亦无争议。
WoW 12.0 封禁了 API，Permes 怎么应对？ +
12.0 对 buff、debuff、技能冷却、充能、战斗状态、副本环境全面封禁 API 并严控 taint。Permes 对 taint 做了全面处理，多技术协同绕过，通过计算机视觉直接识别屏幕状态，12.0 依然稳定运行。
CPU 占用高吗？ +
极低。窄域识别引擎针对 WoW UI 优化，模型不足 1MB，单次推理 <5ms，整体 CPU 占用不足 10%，无需独显。
DPS 水平如何？ +
Permes 内置 SimulationCraft 闭环评估，JSON 规则转 APL 跑分，DPS 达到 SimC 理论值 ±2%。每个技能的次数与占比差距量化可见，迭代到收敛。
支持坦克和治疗吗？ +
完整支持。输出、坦克、治疗三大职责全覆盖。坦克含自动减伤、按血量阈值释放防御技能；治疗含自动焦点打断（不断钢条）、自动选择受伤目标。顺滑插入技能让循环自然流畅。
能否顺滑插入技能？ +
可以。Permes 的规则引擎支持顺滑插入技能，在 GCD 间隙自动插入瞬发技能，不打断主循环节奏，让输出循环自然流畅，最大化 DPS。
Permes Hub 有什么特色？ +
Hub 内置 AI 客服随时答疑，积分系统提供多重福利。一切操作在 Hub 中完成，无需下载解压，启动即用。
