一、认识 Permes

Permes 由两部分组成，两者必须同时运行：

组件 作用
Permes Hub 启动器与管理器：激活码登录、下载/更新执行端、充值续费、积分活动、客服支持
Permes 执行端 实际干活的程序：读取游戏画面（OCR + 像素识别）、按输出循环自动按键

⚠ 使用期间请始终保持 Permes Hub 运行（执行端窗口顶部有常驻提示）。关掉 Hub，执行端会停止工作。

左：Permes Hub 主界面（启动 Permes、充值、设置、积分排行、邀请赚积分、积分抽奖、小游戏等入口）。右：Permes 执行端控制台（启动/停止、安装插件、校准、模型训练、输出循环等功能标签）。

二、快速上手（新用户必读）

完整流程：安装 Hub → 激活 → 启动执行端 → 安装游戏插件 → 进游戏设置 → 校准 →（可选）模型训练 → 开打。

2.1 安装 Permes Hub

  1. 运行安装包 PermesHub_Setup.exe。
  2. 安装向导会提示：安装路径与程序名称将自动伪装，无需手动设置——这是正常现象，不是中毒。
  3. 选择安装盘符，按需勾选"创建桌面快捷方式""开机自动启动""安装完成后立即运行"，点击安装。

2.2 激活与登录

  1. 打开 Hub，输入你购买的激活码完成激活。
  2. 激活后 Hub 主界面会显示：模块范围、到期时间（或"永久有效"）、积分余额。
  3. 激活码与设备绑定，更换电脑需要解绑（见 FAQ）。

2.3 启动 Permes 执行端

  1. 点击 Hub 主界面的绿色大按钮 启动 Permes。
  2. 首次启动会自动下载执行端，等待进度条走完即可。
  3. 执行端启动后是一个控制台窗口，顶部一排功能标签：
    安装插件｜校准坐标｜模型训练｜输出循环｜生存/治疗设置｜切换模块｜热键设置｜SimC 模拟。

如果你使用罗技鼠标，可点击 Hub 的 GHUB 按键保护 按钮，它会自动打开 Logitech G HUB 配合按键保护。

2.4 安装游戏插件

  1. 在执行端点击 安装插件，弹出"选择要安装的职业"对话框。
  2. 确认目标目录指向正式服的 _retail_ 文件夹（例如 D:\World of Warcraft\_retail_）——这是最常见的出错点，选错目录插件不会生效。
  3. 勾选你玩的职业（默认全选），点 OK。

插件安装后在游戏插件列表里显示为一串随机字符名（例如 P3F8847DE），这是刻意的伪装，找不到"Permes"字样是正常的。

2.5 切换模块（选职业专精）

进游戏后第一件事：在执行端点 切换模块，选择你当前角色的职业 + 专精，点确定。

⚠ 玩什么号就选什么专精。选错专精，输出循环、按键、识别区域全都不对。

三、游戏内首次设置

3.1 启用插件

  1. 按 ESC 打开游戏菜单 → 点 插件（AddOns）。
  2. 勾选顶部的 加载过期插件。
  3. 确认 Permes 插件（随机名，如 P3F8847DE）已勾选；如有插件标红"Requires Reload"，点 重载界面（Reload UI）。


3.2 开启冷却管理器（Cooldown Manager）

Permes 通过游戏自带的冷却管理器读取技能冷却和 Buff 层数，必须正确设置：

  1. 按 ESC → 编辑模式（Edit Mode），勾选显示 冷却管理器。
  2. 选中"重要技能冷却"框体，建议设置：
  3. 方位：水平；图标方向：右
  4. 图标尺寸 ≥ 110%（推荐 120% 以上）——图标太小会识别不到
  5. 可见性：始终可见；勾选 显示倒计时、显示说明条
  6. 选中"追踪的增益效果"框体，同样设置，并且：
  7. ⚠ "未激活时隐藏"一定要取消勾选，否则 Permes 读不到 Buff 数字。



3.3 导入官方布局

如果你的冷却管理器布局和官方要求不一致（屏幕上会弹出提示条"检测到冷却管理器缺少下列技能"）：

  1. 不要自己手动摆框体，直接点击游戏内 Permes 悬浮菜单的 官方布局（或提示条上的"导入官方布局"）。
  2. 游戏会自动 reload 一次。
  3. ⚠ 导入后不要移动 Permes 需要读取的那些图标——像素识别按固定位置取图，挪了位置就识别不到。

3.4 校准坐标

校准是让执行端知道"屏幕上哪个位置放着哪个数字"的过程：

  1. 在执行端点击 校准坐标（或点游戏内悬浮菜单的"校准"）。
  2. 游戏聊天框会输出日志：已导出 5 个 ROI 坐标，请执行 /reload。
  3. 在聊天框输入 /reload 重载界面——校准必须 reload 才生效。
  4. 悬浮菜单状态从"校准：待 reload"变为 校准：已完成，即成功。

验证校准是否成功：

  1. 点执行端 ▶ 启动，去木桩区打桩。
  2. 观察执行端"OCR 识别"区域的数字（如腐化层数、天灾打击层数）是否与游戏内 Buff 图标上的数字实时一致。
  3. 一致 = 校准成功；不一致 = 去 第九章 故障排查。

四、AI 模型训练（识别不准时必做）

Permes 的数字识别模型是本地训练、本地运行的。以下情况必须自己训练一次模型：

  • 首次使用（你的分辨率 / DPI / UI 缩放与官方默认环境不同）
  • 更换电脑、显示器、分辨率或游戏 UI 缩放之后
  • 校准成功但 OCR 识别数字仍然错误

训练步骤

  1. 在执行端点击 模型训练。
  2. 如果提示"检测到已有当前模块微调模型"：
  3. 从没训练过自己模型的 → 选 手动重新训练
  4. 已有模型、想继续优化的 → 选 自动/手动加强训练
  5. 进入自动采集：程序每 2 秒截图一次，共 30 张。此期间你要：
  6. 待在木桩区持续战斗；
  7. 确保冷却管理器的图标和 Buff 数字在不断变化（采集的就是这些数字）；
  8. 保持总开关开启，让循环自动打即可。
  9. 采集完进入数据标注：首次使用推荐"手工标注"（"AI 自动标注"是 Beta 功能）。
  10. 逐张检查识别结果（共 30 张）：识别错的直接在输入框里改成正确数字；没有数字的区域改成空。
  11. ⚠ 一个 ROI 白框里不能框到两个数字，否则 AI 会识别成两位数——布局有问题先回去调布局。
  12. 检查完毕开始训练，训练完成后执行端会自动重启——这是正常现象。

五、日常使用

5.1 游戏内悬浮菜单

游戏画面左侧的竖排悬浮菜单是日常操作中心：

菜单项 作用
总开关 绿=开 / 红=关。开启后 Permes 按输出循环自动按键。建议长期保持打开
爆发 爆发技能开关，团战/爆发期打开
自/单/群 模式切换：自动 / 单体 / 群体
设置 打开插件设置面板（见 5.2）
隐藏 隐藏菜单
校准 显示校准状态，点击重新校准
官方布局 一键导入官方界面布局

5.2 插件设置面板（悬浮菜单 → 设置）

① 设置页

  • 总开关 / 爆发开关 / 模式切换的按键绑定（默认总开关、爆发绑定在鼠标侧键 BUTTON4/BUTTON5）
  • 复选项：自动寻怪、显示调试信息、禁用文字提示、禁用音效
  • 底部：SimC 导入（见 6.4）、恢复默认设置

② 一键导入天赋页

列出官方推荐天赋方案（如 1 分钟爆发轴 / 30 秒爆发轴两套），选中后点 Apply 应用。

③ 专属按键宏页

当前专精每个技能的按键绑定列表，点技能右侧的 Edit 可修改。这里的按键必须与你在游戏里的实际键位一致，Permes 才能按对技能。

④ CVar 设置页（游戏参数优化）

  • 延迟容限（SpellQueueWindow）：推荐 150
  • 参数表逐项列出 当前值 / 目标值 / 状态，状态全绿"安全"即正常
  • 直接点 设置为推荐值 一键优化即可

⑤ 制作按键宏页

通用宏命令，可在聊天框直接输入生效，也可复制进游戏宏、绑定按键：

命令 作用
/ON 总开关开
/OFF 总开关关
/bfk 爆发开
/bfg 爆发关
/dan 单体模式
/qun 群体模式
/zi 自动模式
/charu 临时关闭总开关 0.5 秒（插入手动技能用）

/charu 典型用法——把鼠标指向类手动技能写进宏，避免和自动循环抢按键：

/charu /cast [@mouseover,harm,exists]死亡之握;死亡之握

/charu /cast [@player]反魔法领域

六、进阶功能

6.1 输出循环（Rotation Rules）

执行端 → 输出循环，左侧是"动作源"（技能→按键映射），右侧是"输出循环流水线"：

  • 规则从上到下按优先级执行，每条规则可附带触发条件（如"符文 ≥ 4""目标距离 ≤ 5""自身血量 ≤ 30"）。
  • 普通用户：直接点"导入"选官方规则（如【官方】系列优化版），不需要自己改。
  • 会改的用户：可调整顺序、增删规则、为技能添加条件；改完点 保存到当前规则；规则支持 导入/导出 本地备份。

示例：给"灵界打击"添加条件 自身血量 ≤ 30% 且 符文能量 ≥ 30，即可实现低血量自动回血。

6.2 生存/治疗设置

执行端 → 生存/治疗设置。所有选项默认关闭，勾选才生效：

  • 自动打断：打断当前目标 / 打断焦点目标，可设延迟（1 loop = 500ms；建议 0.25 秒左右，模拟人工、避免秒断）
  • 减伤/消耗品阈值：血量低于设定百分比自动释放（如 反魔法护罩 ≤60%、冰封之韧 ≤35%、红药水 ≤50%、治疗石 ≤30%）
  • 治疗设置：按像素条血量自动焦点重伤队友（自动跳过坦克）；重伤阈值默认血量低于 20%

改完点 保存。

6.3 热键设置

执行端 → 热键设置：全局启动热键（默认 CTRL+P，启动循环+爆发）、停止热键（默认 CTRL+O）。

6.4 SimC 模拟

  1. 下载 SimulationCraft（SimC）软件。
  2. 游戏内插件设置面板 → 设置页 → SimC 导入。
  3. 导出 SimC 字符串后，游戏内执行 /reload 保存到 SavedVariables。

八、使用前环境检查清单

识别不准、循环不对，先过一遍这份清单，90% 的问题出在这里：

  • [ ] 游戏内渲染倍数 = 100%（选项 → 系统 → 图形）
  • [ ] 显卡控制面板的渲染比例也 = 100%（NVIDIA App/控制面板 → 程序设置 → WoW，两处必须一致）
  • [ ] Windows 显示缩放 = 100%、分辨率为推荐值（设置 → 系统 → 屏幕）
  • [ ] 多显示器：每一块屏都检查缩放与分辨率
  • [ ] 冷却管理器已开启，图标尺寸 ≥ 110%（推荐 120%+）
  • [ ] "追踪的增益效果"的 "未激活时隐藏"已取消勾选
  • [ ] 没有任何 UI 窗口遮挡监控区域（角色面板、背包等挡住 PixelBar/冷却数字时，Permes 就"看不见"了）
  • [ ] CVar 设置页各项状态全绿"安全"（不确定就点"设置为推荐值"）
  • [ ] 悬浮菜单显示 校准：已完成
  • [ ] 执行端"切换模块"选的是当前角色的专精

九、故障排查

9.1 自检：游戏状态与 OCR 核对

打开执行端主窗口，对照游戏实际状态检查两个区域：

  • 游戏状态区：总开关、爆发、符文、符文能量、目标血量等是否与游戏一致；
  • OCR 识别区：打木桩时识别到的 Buff 层数/冷却数字是否与游戏内图标上的数字一一对应。

哪一项对不上，问题就出在哪一项：游戏状态不对 → 校准/布局问题；OCR 数字不对 → 校准或模型问题。

9.2 重新校准标准流程

  1. 执行端点 停止。
  2. 点悬浮菜单的 校准（或执行端"校准坐标"）。
  3. 聊天框出现 已导出 5 个 ROI 坐标 后，执行 /reload。
  4. 确认悬浮菜单变回 校准：已完成，被移动过的监控图标拖回正确位置。
  5. 点 启动，打木桩重新验证。

9.3 循环"卡技能"排查法

某个技能冷却好了却一直不放：

  1. 打开 输出循环，找到该技能的规则，检查触发条件是否写错（条件永远不满足就会卡住）。
  2. 验证方法（二分定位）：
  3. 把疑似规则的条件全部删掉，或新加一条无条件的同名规则；
  4. 如果无条件规则被疯狂连续触发 → 说明原规则的条件有问题；
  5. 定位完成后删掉临时规则。

9.4 手动校准与模型问题

  1. 执行端 → 高级设置（顶部红字"⚠ 不懂别动"，非排错不要动）→ 手动校准。
  2. 手动校准后 OCR 仍然不对 = 识别模型的问题 → 回到 第四章 模型训练 重新训练。

9.5 常见错误速查

现象 原因与处理
插件装了但游戏里找不到 安装目录没指向 _retail_；插件名是随机伪装名（如 P3F8847DE），不是"Permes"
进游戏插件不生效 插件列表勾选"加载过期插件"并 Reload UI
OCR 数字全是 -- 或错乱 先过第八章检查清单，再重新校准，最后训练模型
识别出两位数 ROI 白框框到了两个数字，调整布局使每个监控区只含一个数字
Buff 数字读不到 "追踪的增益效果"勾选了"未激活时隐藏"，取消勾选
图标突然全都不识别 移动过冷却管理器图标 / 改过 UI 缩放，重新导入官方布局 + 重新校准
训练完执行端自己重启 正常现象
执行端提示保持 Hub 运行 Hub 被关了，重新打开 Hub

十、常见问题 FAQ

以下整理自"帮助与诊断"知识库，按场景分类。

激活码与授权

Q：提示"机器码不匹配"？
更换了电脑或主要硬件导致。联系客服核对并解绑旧设备，然后在当前设备重新输入原激活码。

Q：换新电脑提示"设备绑定数量已达上限"？
直接在新电脑上重新激活，系统会自动解除最旧一台设备的绑定；想保留某台旧设备，先在其他设备上退出登录；仍无法激活联系客服清理旧设备。

Q：提示"激活码不存在 / 已被使用"？
核对是否输错（多空格、字母数字混淆），确认来自官方渠道；确认无误仍提示的，联系客服核对卡密状态。

Q：提示"激活码已过期 / 已吊销 / 待审批 / 设备被封禁"？
过期 → 在 Hub 续费或重新购买；待审批（活动/邀请赠送的卡）→ 联系客服或邀请人催审批；吊销/封禁 → 联系客服核实原因。

Q：Hub 未激活 / 没有活跃 License？
在 Hub 重新输入激活码刷新授权，并把 Hub 更新到最新版本。

Q：登录状态过期 / 被登出？
检查网络后重启 Hub；确认同一激活码没有在其他设备同时运行（多点登录会互踢）。

Q：能试用吗？
试用每台设备、每个网络环境仅一次机会；当前试用功能整体暂停，请直接使用正式激活码。

网络与更新

Q：提示"签名无效 / 时间戳无效"，或所有功能突然全部报错？
把 Windows 系统时间设为自动同步（设置 → 时间和语言 → 自动设置时间），关闭代理/VPN，然后重启 Hub。

Q：提示"无法连接服务器 / 请求超时"？
确认能正常上网；关闭代理/VPN；把 Hub 加入防火墙和杀毒软件白名单后重启。提示"维护中"的等几分钟再试。

Q：提示"请求过于频繁"？
等一分钟左右再操作；公司/网吧等共用网络环境换手机热点试试。

Q：下载失败 / 更新包校验不通过？
直接重试让 Hub 重新获取下载链接；换网络（如手机热点）避开劫持；反复失败的，记下提示中方括号里的错误码，通过"帮助与诊断"发给客服。

付费续费

Q：无法创建支付订单？
检查是否有下单未付款的订单（会占用下单额度，约 30 分钟自动过期）；已是永久授权的无需再充值；反复失败联系客服。

Q：付款后没到账？
先在支付宝/PayPal 账单确认是否真的扣款成功。已扣款：在付款页面等系统确认，一般几分钟到账，切勿重复付款；超 30 分钟未到账，联系客服并提供订单号。未扣款：重新下单即可。

Q：主支付宝失败切备用通道的提示？
主通道失败按页面提示改用备用支付宝；主订单已付款的回 Hub 等激活，不要再用备用通道付一次；状态无法确认时先查支付宝账单，切勿重复付款。

Q：PayPal 付款失败/取消？
已取消或失败的回 Hub 重新下单；已付款的回 Hub 等确认，不要重复付款。

模块与专精

Q：提示"无权使用该专精"或专精灰色？
未购买 → 在 Hub 加购该专精后重启 Permes；刚买过 → 重启 Hub 刷新授权。提示"模块已下架"的，更新 Permes 后重新选择模块，旧校准数据会保留。

活动与积分

Q：赛季卡抢购问题？
提示人数较多立刻多点几次；售罄/未开始等下一场；提示已抢购成功无需重复操作。

Q：积分不足 / 抽奖解锁失败？
通过邀请好友、参加活动攒积分；积分扣了但奖励没到账的联系客服。

Q：无法创建邀请？
当前激活码剩余有效期需 ≥ 15 天，不够先续费；邀请生成的激活码需客服审批后生效。

Q：AI 客服使用受限？
每小时最多提问 5 次，达到上限等一小时；提问带上完整错误提示和操作步骤。

Q：修改名字失败？
排行榜名字每 30 天只能修改一次。
